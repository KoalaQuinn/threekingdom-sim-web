# ⚔️ 三国立志传：草莽英雄

纯 JavaScript 文字放置游戏，黄巾之乱开局，你只是一介草莽，在乱世中努力活下去，活到诸侯讨董就是胜利。

## 游戏背景

- 时间：**中平元年 184年** → **初平元年 190年**，总计 84 个回合
- 身份：流落乡野的平民，没有金手指，首要目标：**活下去**
- 结局：活到 190 年 12 月胜利，根据坞堡等级给结局评价；饿死了就是 game over

## 玩法

### 核心循环

- 每个月固定 **3 点行动力**
- 每次操作消耗 1 点，打完点击「结束本月」结算
- 自动扣除粮食消耗（坞堡+私兵），产出资源，推进到下个月

### 我的坞堡

- **主角三属性**：
  - 武力：影响开垦战斗检定
  - 智力：影响政务收益
  - 魅力：影响流民招募、事件说服
  - 每个属性可以升级，每级 +5 点属性值

- **坞堡升级**：
  | 等级 | 名称 | 升级消耗 | 效果 |
  |------|------|----------|------|
  | 1 | 破败草庐 | 初始 | 每月消耗 10 粮食，产出 15 粮食 |
  | 2 | 乡间篱落 | 500 钱 + 500 粮 | 每月消耗 30 粮食，产出 60 粮 10 钱，解锁募兵 |
  | 3 | 坚固坞堡 | 2000 钱 + 3000 粮 | 每月消耗 100 粮食，产出 200 粮 50 钱，贼寇防御加成 |
  | 4 | 乱世壁垒 | 5000 钱 + 8000 粮 | 每月消耗 300 粮食，产出 500 粮 200 钱，随机流民投奔 |

- **坞堡行动**：
  - 亲自开垦：消耗 1AP，立即获得粮食，受武力加成
  - 挑灯夜读：消耗 1AP，增加智力经验，概率升级智力

### 州郡街市

- **官署**：接徭役任务，智力检定，成功给更多钱
- **酒馆**：
  - 50% 概率刷出 NPC（刘备/张飞/华佗 MVP 三个）
  - 花钱请客涨好感，100 好感解锁羁绊技能
  - 50% 概率触发随机文字事件，选项检定
- **市集**：买卖粮食，救急用

### 天下大势

- 经历日志：所有事件自动记录
- NPC 羁绊列表：显示好感和解锁状态

### 离线积累

关闭网页后，时间依然在积累，最多存 12 个月，上线手动领取，不用担心关网页亏进度。

## 结局判定

- **失败（Game Over）**：粮食变成负数 → 饿死，游戏结束
- **胜利（通关）**：活到 190 年 12 月 → 结算结局

## 如何运行

直接浏览器打开 `index.html` 就能玩，不需要后端。

或者启动本地服务器：

```bash
python3 -m http.server 8000
# 访问 http://localhost:8000
```

## 存档

所有数据存在浏览器本地 localStorage，清缓存会丢失，重要进度请自行备份。
