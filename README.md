# ⚔️ 三国模拟器

纯 JavaScript 编写的三国鼎立放置模拟游戏，三分归一统，挂机就能玩。

## 玩法介绍

- **回合制**：每个月前进一次，手动点击
- **离线积累**：关闭网页后，下次打开可以领取积累的资源，最多存 12 个月
- **三国鼎立**：魏蜀吴各据一方，发展人口，产出资源
- **城池建设**：每个城池自动产出资源，人口增长
- **武将**：收集武将，三国著名武将登场，各有属性
- **战争**：发起战争，攻占城池，统一天下

## 初始布局

- **魏**：许昌、洛阳 —— 曹操
- **蜀**：成都、汉中 —— 刘备
- **吴**：建业、柴桑 —— 孙权

每个武将拥有：武力、智力、政治、忠诚四项属性

## 如何运行

直接用浏览器打开 `index.html` 就能玩，不需要后端。

或者启动本地服务器：

```bash
# Python 3
python3 -m http.server 8000
# 访问 http://localhost:8000
```

## 存档

所有数据存在浏览器本地 localStorage，清理缓存会丢失，重要进度请保重。
